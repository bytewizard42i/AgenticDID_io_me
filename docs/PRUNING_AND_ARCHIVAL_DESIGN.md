# Pruning and Archival Protocol

This repository does not define an independent pruning and archival protocol.

The sole source of truth is the [DIDzM Pruning and Archival Protocol](../../DIDzMonolith-docs/standards/DIDZM_PRUNING_AND_ARCHIVAL_PROTOCOL.md).

## AgenticDID responsibility

AgenticDID expresses narrowly scoped authority for an agent to prepare, verify, retrieve, or temporarily materialize an archive on behalf of an authorized principal.

AgenticDID does not create authority, choose retention periods, operate archival storage, or redefine the root protocol. Every agent action must remain bounded by the principal's underlying rights, purpose, scope, and expiration.
