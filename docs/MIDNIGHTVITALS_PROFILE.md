# AgenticDID MidnightVitals Profile

The canonical protocol is maintained at
[`../../DIDzMonolith-docs/standards/MIDNIGHTVITALS_PROTOCOL.md`](../../DIDzMonolith-docs/standards/MIDNIGHTVITALS_PROTOCOL.md).

The AgenticDID profile should verify scoped grants, attenuation, limits, expiry,
counterparty binding, exercise, audit, and revocation without exposing the
principal or private grant contents. This file is a pointer and profile
boundary, not a competing protocol specification.
