# Migrated from `AgenticDID_io_me_MAIN`

This `nerds-only/` folder is the canonical home for the deep protocol /
specification documentation that originally lived in
`bytewizard42i/AgenticDID_io_me_MAIN/docs-nerds-only/`.

It was migrated here so the active `AgenticDID_io_me` repository carries the
full intellectual brain trust forward.

- **`ISSUER_USE_CASES.md`** / **`FIRST_THREE_ISSUERS.md`** / **`ISSUERS_AND_AGENTS_CHART.md`**

## Why "nerds only"

These are the protocol-research and architecture-deep-dive docs. They are not
required reading for users of the `AgenticDID` web app, SDK, or verifier API;
they are required reading for builders extending the protocol or composing it
into related systems (KYCz, DIDz, MidnightVitals, etc.).
